About

AI Financial Advisor is a practical web application that combines user financial inputs, backend analysis, and machine learning prediction to provide a complete personal finance health snapshot.

Its purpose is to make financial planning simpler for users who want to track expenses, estimate future savings, and receive actionable advice without requiring deep financial expertise.

Problem & Solution

Real-World Problem

Many people struggle to convert raw income and expense numbers into meaningful decisions. Financial tools are often too complex, fragmented, or not personalized.

How This Project Solves It

The project transforms user inputs into predictions, risk indicators, and AI-backed advice in one interface, helping users quickly identify overspending and improve financial health.

Features

Financial Analysis

Analyzes income, fixed costs, variable costs, and lifestyle score for health assessment.

AI Recommendations

Generates practical suggestions to reduce risk, optimize spending, and reach savings goals.

Data Processing (CSV)

Uses structured historical CSV data as training input for meaningful financial prediction.

Insights & Predictions

Provides predicted savings, expense ratio insights, and trend-focused decision support.

How It Works

Step 1

Input

User submits income, expense breakdown, savings goal, and lifestyle preferences.

Step 2

Processing

Backend validates and transforms data for analytics and model-ready feature preparation.

Step 3

ML Prediction

Scikit-learn model predicts savings and supports financial health evaluation.

Step 4

Output

App returns health score, overspending risk status, and personalized recommendations.
